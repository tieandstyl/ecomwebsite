## üìä Data Calculations

### Revenue Calculation
```python
Total Revenue = Sum of all CONFIRMED payments
(Excludes pending and failed payments)
```

### Average Order Value
```python
AOV = Total Revenue √∑ Number of Orders
```

### Customer Lifetime Value
```python
CLV = Total Revenue √∑ Number of Customers
```

### Repeat Customer Rate
```python
Repeat Rate = (Customers with >1 order √∑ Total Customers) √ó 100
```

### Category Performance
```python
Category % = (Category Revenue √∑ Total Revenue) √ó 100
```
