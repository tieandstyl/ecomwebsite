## Color Guide

### Gradient Cards (Top Row):
1. **Total Revenue**: Pink → Red gradient (`#f093fb → #f5576c`)
2. **Total Orders**: Blue → Cyan gradient (`#4facfe → #00f2fe`)
3. **Total Customers**: Green → Cyan gradient (`#43e97b → #38f9d7`)
4. **Products Sold**: Pink → Yellow gradient (`#fa709a → #fee140`)

### Charts:
- **Revenue Line**: Purple line with light purple fill (`#667eea`)
- **Doughnut/Pie**: Multi-color (Purple, Pink, Blue, Green)
- **Bars**: Status-based (Orange, Green, Red)

### Inventory Alerts:
- 🔴 **Critical**: Red background (`#fef2f2`), Red border (`#ef4444`)
- 🟡 **Warning**: Orange background (`#fffbeb`), Orange border (`#f59e0b`)
- 🟢 **OK**: Green background (`#f0fdf4`), Green border (`#10b981`)

### Overall Theme:
- **Background**: Light gray (`#f9fafb`)
- **Cards**: White with shadows
- **Text**: Dark gray (`#1f2937`)
- **Accents**: Purple (`#667eea`)

## Interactive Features

1. **Hover Effects**: All cards and charts respond to hover
2. **Click Charts**: Chart.js interactive tooltips
3. **Date Selector**: Dropdown filters entire page
4. **Export Button**: Triggers print dialog
5. **Responsive**: Adapts to screen size
